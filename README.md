
# 💰 Expense Tracker Pro - Indian Edition 🇮🇳

A beautiful console-based Expense Tracker written in Java with Indian Rupee (₹) support.

## ✨ Features

- ✅ Add Income/Expense with Indian Rupees (₹)
- ✅ Beautiful UI with colors and borders
- ✅ 30+ Indian-specific categories (EMI, Mobile Recharge, etc.)
- ✅ View all transactions in formatted table
- ✅ Financial summary with savings rate
- ✅ Category breakdown with progress bars
- ✅ Search transactions
- ✅ Export reports to text file
- ✅ Hindi motivational messages

## 🚀 How to Run

```bash
# Compile
javac ExpenseTrackerPro.java

# Run
java ExpenseTrackerPro
